    Présentation golbale de notre cellule

    Toutes les deux semaines, le mercredi matin de 10h30 à 12h30, les membres de la Cellule BioInfo de Saint Louis sont mobilisés pour apporter un soutien direct à tous les membres de l’hôpital Saint Louis qui recherchent de l'aide sur des sujets liés à la bio-informatique. Vous pouvez venir directement en salle de réunion du Service de Génétique Neurovasculaire (l’accès se fait par le Carré historique, porte 13 – 1er étage) ou nous contacter via notre Slack.

    Les membres de la Cellule BioInfo peuvent vous aider dans les domaines suivants (liste non exhaustive)

  • Révision de code (Bash, Python, R)
  • Développement de bases de données
  • Développement de logiciels
  • Aide à l’analyse de données OMICS
  • Développement d'outils Web

    • Venez avec votre ordinateur perso et/ou de travail et vos données, ainsi que votre code si nécessaire.
